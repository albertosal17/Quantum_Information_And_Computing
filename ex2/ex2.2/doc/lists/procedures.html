matmul_method1 matmul_methods Function

This functions compute the product matrix of two square matrices using the standard algorithms of performing the multiplication "row-by-column". The elements in position (i,j) of the product matrix is computed as a scalar product between the row i of the left-most matrix and the column j of the right-most one.
matmul_method2 matmul_methods Function

This functions compute the product matrix of two square matrices using the transpose of the left-moste one. In this way, it allow to make the product as the sum of scalar products between the columns of the two original matrices. This is beneficial in Fortran as by default it stores arrays columns in contiguos location of memory, making it easier to retrieve the entries involved in the multiplication.
matmul_method3 matmul_methods Function

This functions compute the product matrix of two square matrices using the built-in method of Fortran.
